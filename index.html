Hello, this is the project I've been working on throughout the term.

Java Project 2021: Adventure Time

The object of the game is to collect as many Cinnamonbuns that spawn in. However, while doing so, you will have to kill sone enemies along the way in order to proceed to the next level. To move around, you would use the arrow key and to attack you would press spacebar. On the right side, there a GUI menu, which you can save the level you are working. Once complete you can save your score on the leaderboards.

Check out the code on Github: https://Farid48.github.io/
